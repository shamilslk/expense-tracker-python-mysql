# Expense Tracker Dashboard

A desktop-based Expense Tracker Dashboard built using Python, Tkinter, MySQL, and Matplotlib.

## Features

* Add, Update, Delete, and View expenses (CRUD)
* MySQL database integration
* Category-based filtering
* Expense sorting
* Live search functionality
* Statistics Dashboard:

  * Total Expenses
  * Number of Expenses
  * Maximum Expense
  * Average Expense
* Donut Chart visualization using Matplotlib
* CSV Export functionality

## Technologies Used

* Python
* Tkinter
* MySQL
* Matplotlib

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
pip install mysql-connector-python matplotlib
```

3. Create the MySQL database and table

4. Update database credentials in the Python file

5. Run the application

```bash
python main.py
```
